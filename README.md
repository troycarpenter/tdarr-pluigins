# tdarr-pluigins
Tdarr plugins written specifically for my environment

Designed for Jellyfin, not Plex.
Optimized for OTA recordings and downloaded media.
Uses HandBrake NVENC for transcoding.
Preserves broadcast closed captions when available.
Skips HDR downloaded movies.
Library-specific policy (Downloaded TV, Downloaded Movies, Recorded TV, Recorded Movies, Test).
Built as small, composable Tdarr plugins rather than one large plugin.
